# AI-Powered 5G Open RAN Optimizer

An AI-based optimization framework for 5G Open RAN networks that predicts traffic load and suggests resource allocation strategies for base stations.

## Overview

This project explores how machine learning can be applied to optimize resource allocation in 5G Open RAN environments.

The system analyzes historical network traffic data to:

- Predict future traffic load
- Detect abnormal network behavior
- Suggest resource allocation adjustments for base stations

The goal is to improve network efficiency and support intelligent network management in modern 5G infrastructures.

---

## Features

- **Traffic Prediction**
  - Predicts future network traffic load for each cell.

- **Network Anomaly Detection**
  - Detects abnormal traffic patterns using machine learning models.

- **Dynamic Resource Optimization**
  - Suggests resource scaling strategies based on predicted traffic.

- **Energy Efficiency Awareness**
  - Helps reduce unnecessary resource allocation during low traffic periods.
